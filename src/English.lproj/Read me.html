Taply 1.7.1

Taply is a simple soundfile player. Although a lot of similar tools do exist for Mac OS X, I did not know of any which behaves exactly the way I want it to, so I wrote this application.

The main difference between Taply and other players lies in the fact that it consists of nothing more than a tiny floating window and that it will quit after having finished playing. This way, (at least for me) it is the perfect tool to scan quickly through a large number of audio files.

Using Taply

When doubleclicked, you will see an open panel to choose one or more sound files. Taply will play the selected files (files which it cannot play will be skipped) and quit afterwards.

You can invoke a contextual menu by right-clicking or control-clicking in any unused space in the window, which gives direct access to any file in the queue, removes any files (except the current one) from the queue or quits Taply or lets you invoke the preferences.

If one or more sound file(s) or folder(s) containing any sound(s) is/are dropped onto the window or the onto the application icon, the file(s) is/are added to the end of the playback queue.


Supported file formats

The list of supported audio file formats may depend on the operating system version. On recent OS X versions, at least AIFF, WAV, MP3 and AAC (only without DRM) should be usable.
