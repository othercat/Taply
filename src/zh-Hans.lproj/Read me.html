Taply 2.0.0-dev 现代 macOS 版

Taply 是一款简洁的音频文件播放器。虽然 macOS 上有许多类似的工具，但 Taply 的设计目标是尽可能小巧：一个浮动小窗口，播完自动退出，非常适合快速浏览大量音频文件。

版本更新说明

2.0.0-dev（由 Richard Li 维护）：

  • 将播放内核从 QuickTime 迁移到 AVFoundation，支持现代 macOS SDK 构建
  • 目标平台：Intel macOS 10.14+ / Apple Silicon macOS 11.0+
  • Universal binary（x86_64 + arm64）
  • 移除 Carbon.framework 和 QuickTime.framework 依赖
  • 适配深色模式
  • 新增简体中文界面
  • 更新拖放 API（NSPasteboardTypeFileURL）
  • 更新 deprecated AppKit 常量

使用方法

双击 Taply，会弹出文件选择面板，选择一个或多个音频文件即可开始播放。无法播放的文件会自动跳过，播放完毕后 Taply 会自动退出。

在窗口空白处右键或 Control 点击，可以呼出快捷菜单：直接跳转到播放列表中的任意文件、清空播放列表、打开偏好设置或退出 Taply。

将音频文件或包含音频的文件夹拖放到窗口或应用图标上，文件会被添加到播放队列末尾。

支持的文件格式

支持的音频格式取决于系统版本。在 macOS 10.14 及以上版本，至少支持以下格式：

  • AIFF / AIFC
  • WAV
  • MP3
  • M4A（AAC，无 DRM）
