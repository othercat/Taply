Taply 1.7.1

Taply ist eine simple Abspielsoftware für Audiodateien. Auch wenn es bereits einen ganzen Haufen ähnlicher Programme für Mac OS X gibt, war bzw. ist mir keines bekannt, das sich so verhält, wie ich es gerne hätte, und so wurde Taply geboren.

Der – für meine Anwendungszwecke – wesentliche Unterschied zwischen Taply und anderen Playern besteht darin, dass die Benutzeroberfläche von Taply sich auf ein einziges, schwebendes Paletten-Fenster beschränkt und sich nach Wiedergabe der Datei(en) automatisch beendet. Insofern ist es für mich das ideale Tool, um mir aus dem Finder heraus auf die Schnelle einen Überblick über Audiodateien zu verschaffen.

Benutzung

Wird das Programm durch Doppelklick auf das Programmsymbol gestartet, wird ein Öffnen-Dialog angezeigt, um eine oder mehrere Audiodateien auszuwählen. Die gewählte(n) Datei(en) werden wiedergegeben (Dateiformate, die Taply nicht abspielen kann, werden übersprungen, nachdem eine Warnung angezeigt wurde), und anschließend beendet sich das Programm automatisch.

Über ein Kontextmenü, das durch Rechts-Klick bzw. Control-Klick in einen freien Bereich oder auf das Textfeld aufgerufen werden kann, ist es möglich, direkt zu einer bestimmten Datei zu springen, die Wiedergabeliste zu löschen oder das Programm zu beenden.

Zieht man Audiodateien (oder einen Ordner, der Audiodateien enthält) auf das Fenster oder das Programmsymbol während Taply bereits andere Dateien abspielt, wird bzw. werden die neue(n) Datei(en) an das Ende der Wiedergabe-Warteschlange angehängt.


Unterstützte Dateiformate

Die Liste der unterstützten Dateiformate kann von der jeweiligen Betriebssystemversion abhängen. Auf aktuellen Systemversionen sollten mindestens AIFF, WAV, MP3 und AAC (nur ohne DRM) wiedergegeben werden können.

Preis

Taply ist kostenlos. Aber wenn das Programm für Sie nützlich ist und Sie ein »Danke« senden möchten, gibt es dafür eine Reihe von Möglichkeiten. Auf meiner Website erfahren Sie mehr hierzu.
